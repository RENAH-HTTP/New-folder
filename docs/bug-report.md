# Bug Report

**Edition:** Live 12

**Released:** Novermber 25, 2025

---

**Audio & Playback**

- Fixed bug in Hybrid Reverb where the former reverb tail was still present after the device was switched off.

- Fixed timming issue that might have happend during recording or bouncing if tempo automation was present.

---

**Crashes & Hangs**

- Fixed a crash when initiating some plugins (e.g., SoundBetter's Halo Effect) if Push 2 was connected to the machine. 

- Fixed a bug that made Live crash druing dense MIDI CC recording.

---

**VST / Plug-in Support**

- Improved UI animation of VST3 plug-ins resulting in a smotter animation of parameters.

- Fixed visual glitches when altering parameters of VST plug-in's UI.

---

**MIDI**

- Fixed a bug relating Expression Control's curve controls not working untill switching tabs.

---

**UI & Visual**

- Reduced spacing between track index number and track name in default track names.

---

**Performance & Stability**

- Improved Live's CPU load when moving clips in a Set with multiple locators. 

---

**Control Surfaces**

- Removed device-related info in Live's Status Bar in the Komplete Kontrol S Mk3.

---

**Accessibility**

- Fixed potential crashes in Live while navigating the transport bar using VoiceOver.

---

**Workflow & Arrangement**

- Fixed an focus issue where closing certain views (e.g., Undo History, File Manager - in the Help View) would cause the focus to be lost. Now, focus returns to visibles main view (Arrangement or Session) when one of the aforemantioned views is closed.
